Android 开发初体验

一直以来自己的开发都是基于命令行界面，除了 JavaFX 没有开发过带有图形界面的现代应用程序，一方面是学校的课程没有教授相关内容，另一方面是自己一直没确定自己要做的东西。但是上周听了翁培钧学长的事迹以后，才真正敢确定自己想做的事情，所以这周便尝试开始入门移动 App 开发。

虽然我用的是 macOS，开发 iOS 会更加便利，但是正好由于最近有一个团队 Android 项目需要做，所以选择从 Android 开始我的第一次完整的移动 App 开发体验。

安装 Android Studio 并搭建 Android 开发环境

首先前往 Android Developers 官网 下载 Android Studio 3.5 for Mac 的 dmg 安装包，安装完毕，第一次初始化由于未安装 Android SDK，显示 Unable to access Android SDK add-on list 点击 cancel 忽略。

在选择 Install Type 时选择 Custom 自定义安装，下一步选择下载所有组件，完成安装。

Hello World 项目的构建与打包安装

新建一个 Empty Activity 项目，选择 Minimum API Level 为 API 28 : Android 9.0 (Pie)，进入项目，初次使用需要配置 Run/Debug Configurations，按照下图所示配置完毕：

然后即可按下 ⌃R 运行程序，等待虚拟机开机完毕安装好 App，界面应如下所示：

最后选择 Build –> Build Bundle(s)/APK(s) –> Build APK(s)，得到本地 apk 文件，将该文件发送到 Android 手机上或将手机与电脑连接进行真机调试实际安装，显示正确界面，至此，最简单的 Android Hello World 项目开发完成，耗时10分钟不到，项目非常容易，基于已有模板，仅仅作为对工具的一个尝试。
